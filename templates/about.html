Whois Information

This is a domain whois lookup tool written in Python with the help of Flask. I am a domain investor. I check whois information every day. I've tried lots of whois lookup websites and not so satisfied. So I decide to write one myself.
I trid whois and python-whois module, but none of them satisfy my requirements. I like the whois tool provided by Ubuntu. So I decide to call whois command from shell and receive the output.

Usage

This tool support two usages:
  • Search whois information directly through home page
  • Get whois information with '/whois/your_domain'

Dependency

  • Ubuntu 16.04LTS
  • Python 2.7
  • Flask 0.12.1
  • Jinja2 2.9.6
  • Bootstrap v4-alpha
